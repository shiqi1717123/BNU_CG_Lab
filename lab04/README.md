# Lab04: Phong光照模型

## 实验概述

本实验使用Python与Taichi实现一个基于光线投射的局部光照渲染器。场景不依赖外部模型文件，而是在 Taichi Kernel 中通过隐式几何方程定义球体与圆锥，并在每个像素上完成求交、深度竞争与Phong着色。

首先完成了以下内容：

- 红色球体：圆心 `(-1.2, -0.2, 0)`，半径 `1.2`，基础颜色 `(0.8, 0.1, 0.1)`。
- 紫色圆锥：顶点 `(1.2, 1.2, 0)`，底面高度 `y = -1.4`，底面半径 `1.2`，基础颜色 `(0.6, 0.2, 0.8)`。
- 摄像机固定在 `(0, 0, 5)`，点光源固定在 `(2, 3, 4)`，光源颜色为白色。
- 对每条像素射线分别计算球体、圆锥侧面与圆锥底面的交点，并选择最小正数 `t` 作为最终可见表面。
- 在交点处计算法向量、光线方向、视线方向与反射方向，叠加 Ambient、Diffuse、Specular 三个分量得到最终颜色。
- 使用 `ti.ui.Window` 与 4 个滑动条实时调节 `Ka`、`Kd`、`Ks` 和 `Shininess`。

## 交互说明

窗口左上角提供 Phong 光照参数面板：

- `Ka (Ambient)`：环境光系数，范围 `0.0 ~ 1.0`，默认值 `0.2`。
- `Kd (Diffuse)`：漫反射系数，范围 `0.0 ~ 1.0`，默认值 `0.7`。
- `Ks (Specular)`：镜面高光系数，范围 `0.0 ~ 1.0`，默认值 `0.5`。
- `Shininess`：高光指数，范围 `1.0 ~ 128.0`，默认值 `32.0`。

拖动滑动条，我们可以实时观察到环境光、漫反射和镜面高光对物体外观的影响。
环境光可以提供底光并且调节亮度，漫反射系数决定物体表面漫反射的程度，镜面高光主要和物体集中反射的白色区域有关，这里光线几乎被镜面反射，高光指数可以修改高光的覆盖范围。

选做版本加了两个复选框：

- `Use Blinn-Phong`：启用后使用半程向量 `H` 计算镜面高光；关闭后使用原始 Phong 反射向量 `R`。
- `Use Hard Shadow`：启用后从交点向光源发射阴影射线；若光源前方被其他几何体遮挡，则该点只保留环境光。

版本还加入了一个由代码定义的地面平面，作为阴影接收面。这样让硬阴影的显示更加明显和直观。

## 实现要点

### 1. 光线生成

程序将每个像素映射到相机前方的成像平面，生成从摄像机位置 `(0, 0, 5)` 出发、朝向场景的单位方向向量。这样每个像素都对应一条独立射线。

### 2. 几何求交

球体求交使用标准二次方程：

```math
\|O + tD - C\|^2 = r^2
```

圆锥侧面使用以顶点为局部原点的隐式方程：

```math
x^2 + z^2 = k^2 y^2
```

其中 `k = base_radius / height`。求出候选交点后，还会检查交点是否位于顶点和底面之间。圆锥底面使用平面求交，并检查交点是否落在底面圆盘内。

### 3. 深度竞争

每条射线会分别测试球体、圆锥侧面和圆锥底面。如果同时打中多个物体，就选择距离摄像机最近的交点，也就是最小的正`t`。这相当于光线投射中的Z-buffer深度竞争的逻辑，可以保证遮挡关系正确。

### 4. Phong着色

命中表面后，程序利用phong计算：

```math
I = I_{ambient} + I_{diffuse} + I_{specular}
```

其中：

```math
I_{ambient} = K_a C_{light} C_{object}
```

```math
I_{diffuse} = K_d \max(0, N \cdot L) C_{light} C_{object}
```

```math
I_{specular} = K_s \max(0, R \cdot V)^n C_{light}
```

写入像素前会将颜色约束到 `[0, 1]`范围内，避免过曝。

## 选做部分

### 1. Blinn-Phong高光

Phong模型使用理想反射方向 `R` 与视线方向 `V` 的夹角计算高光：

```math
I_{specular} = K_s \max(0, R \cdot V)^n C_{light}
```

Blinn-Phong模型不直接计算反射方向，而是改用光线方向 `L` 与视线方向 `V` 的半程向量：

```math
H = \frac{L + V}{\|L + V\|}
```

```math
I_{specular} = K_s \max(0, N \cdot H)^n C_{light}
```

在相同的高光指数`Shininess` 下，Blinn-Phong的高光通常更稳定，尤其是在较大入射角处，高光边缘不容易像Phong那样快速消失；视觉上会显得更宽、更连续。Phong高光更依赖反射方向与视线方向的精确对齐，因此高光区域可能更集中，对边缘变化更敏感（可以参考demo里面的演示视频，当进度条拉到最左端时，phong模型的高光会发生一种突变，但是blinn-phong版本变化更平滑和柔和）。

### 2. 硬阴影

选做版本在主射线命中表面后，会从交点沿光源方向发射一条阴影射线：

```math
P_{shadow}(t) = P + \epsilon N + tL
```

其中 `epsilon` 是很小的偏移量，用来避免阴影射线刚发出时立刻再次命中当前表面。如果阴影射线在到达光源之前先命中球体或圆锥，说明该点被遮挡，此时只计算环境光：

```math
I = I_{ambient}
```

由于硬阴影没有半影过渡，所以阴影边界会比较清晰比较硬（所以叫硬阴影）。这和真实光源有面积时产生的软阴影不同，但实现简单，而且能直观体现二次射线检测遮挡关系的思想。

## 实验总结

本次实验之后，我们可以把光线求交、局部光照模型等串联起来理解。调节滑动条时，`Ka` 主要控制整体亮度下限，`Kd` 决定受光面的主体明暗变化，`Ks` 与 `Shininess` 则共同控制高光的强度和集中程度。选做部分进一步加入了半程向量高光和阴影射线，让我们看到了改进算法后渲染质量的提高。
